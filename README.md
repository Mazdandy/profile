# QA Engineer Landing Page

This project is a personal landing page for Dwi Fajar Dandy Saputra. It presents a QA-focused portfolio with a playful visual style, highlighting professional background, work experience, technical skills, and contact information.

## Overview

The site is built as a lightweight static website using plain HTML, CSS, and JavaScript. It is designed to be simple to deploy on GitHub Pages while still providing a polished interactive experience.

## Features

- Hero section with animated typography and call-to-action buttons
- About section introducing the profile and QA background
- Work experience section covering QA Engineer and IT Automation roles
- Skills section listing testing and development tools
- Contact section with email and LinkedIn links
- Smooth scrolling and scroll-triggered animations

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

- [index.html](file:///Users/dfdandys/Documents/Project/landing%20pages/index.html): page structure and content
- [styles.css](file:///Users/dfdandys/Documents/Project/landing%20pages/styles.css): layout, colors, typography, and animations
- [script.js](file:///Users/dfdandys/Documents/Project/landing%20pages/script.js): smooth scrolling and reveal interactions

## Run Locally

Open [index.html](file:///Users/dfdandys/Documents/Project/landing%20pages/index.html) directly in a browser, or serve the folder with a local static server.

Example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This project is suitable for GitHub Pages because it is a static site with no backend dependencies.

## Purpose

The goal of this project is to provide a clean and memorable personal web portal that showcases QA expertise, work history, and ways to get in touch.
